# AI Business Agent - Flutter Mobile Application

AI-агент для бизнеса с каталогом товаров, системой заказов, FAQ и поддержкой.

## Особенности

- 🛍️ **Каталог товаров** - просмотр и заказ услуг
- 📋 **Система заказов** - оформление и отслеживание заказов
- 💬 **AI-поддержка** - чат с AI-ассистентом
- ❓ **FAQ** - ответы на часто задаваемые вопросы
- ⚙️ **Админ-панель** - управление заказами и товарами

## Технологии

- **Flutter** - UI фреймворк
- **SQLite** - локальная база данных
- **Provider** - управление состоянием
- **Dart** - язык программирования

## Дизайн

- **Цвета кнопок:**
  - Green: `#00FF00` - основные действия
  - Gray: `#CCCCCC` - второстепенные действия
  - Red: `#FF4444` - удаление/отмена
- **Стиль кнопок:** полукруглые (`StadiumBorder`) с эмодзи
- **Фон:** белый
- **Тон:** дружелюбно-деловой с эмодзи

## Автоочистка

Приложения автоматически удаляет заказы старше 30 дней при запуске.

## Админ-панель

Доступ по ПИН-коду `1234`. Админ может:
- Просматривать все заказы
- Менять статусы заказов
- Очищать старые заказы вручную

## Установка

1. Установите Flutter: https://flutter.dev/docs/get-started/install
2. Клонируйте репозиторий
3. Выполните `flutter pub get`
4. Запустите приложение: `flutter run`
